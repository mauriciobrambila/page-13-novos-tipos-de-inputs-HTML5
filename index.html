Novos inputs no HTML 5


      Neste artigo vamos tratar sobre HTM5 e os novos input types que surgiram com ela.

      Agora assim não precisamos mais criar validações em JavaScript para campos numéricos, e-mails, datas,

    entre outros que estaremos vendo ao decorrer do artigo,foram adicionados mais 13 novos tipos de inputs à HTML5,

 porém como sempre temos problemas de compatibilidade.

 Nem todos os navegadores suportam todos os novos input types.

Entretanto podemos começar a usá-los,

se por acaso eles não forem suportados, irão se comportar como um campo de texto comum.

          Ao todo temos 23 valores possíveis para o atributo type dos inputs, incluindo os que já existiam e mais os treze novos que surgiram com o HTML5.
